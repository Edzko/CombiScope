Connecting to the CombiScope

 

The PM33XX software communicates with the Fluke CombiScope via a Serial port interface (RS232). Select the connected port and applicable baudrate from the drop-down menus in the user interface, and click "Connect".

When the software successfully connects, it will display information about the CombiScope in the status bar, and the icon on the "Connect" button with change to an image of two connectors plugged.

It the icon still shows the connectors unplugged, then connection was not successful.

Troubleshooting

NOTE: The most common cause for failure when connecting to the CombiScope is related to the assigned port number. When the connection fails, and the serial port identifier is a double-digit number, then try to re-assign a single-digit port number to the serial interface through the Hardware Devices configuration of Windows.
