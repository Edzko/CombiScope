FlukeView 4.0 Combiscope Acquisition Software

The PM33XX Software Application supports any of the FLUKE PM3382 ~ PM3394 CombiScopes with serial port interfaces. The software can capture screen hardcopies or individual traces of active channels or memory channels. The Cursors and the Info window can be used for further analysys. Similarly, the Fast Fourier Transform feature allows for analyzing the captured trace information in the frequency domain.

Further information in this documentation covers:

  1. Connecting to the CombiScope
  2. Capturing hardcopy and traces
  3. Using Info and Cursors
  4. Using the FFT
  5. The User Interface Window
  6. Additional Features
