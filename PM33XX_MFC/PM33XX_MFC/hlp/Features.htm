Additional Features of the PM33XX Software

Settings

Clipboard formats

A snapshot of the graphical display can be captured by clicking on the "Copy" button, or by simply double-clicking anywhere in the graphical display area. The snapshot can be retrieved in any external Windows application by selecting "Paste As", and choosing a desired image format.

You can also paste the captured data as text, which results in a table of signal values across the time axis. Pasting in Excel will automatically place the values in individual columns.

Printing

Print the current view of the graphical window by selecting the "Print" command from the drop-down application commands list.

Zooming

In order to zoom in any detail of the chart, click on the "Zoom" button in the main application ribbon. Then draw a rectangle around the area of interest. The software will then re-display only the selected area in the full graphical area of the interface window.

Colors

When selecting a signal trace, or a cursor, the color selector button will update with the color of the selection. Then click on the button and select a new color. Colors are retained for the next time when the software is used.

Click anywhere in the background of the chart area, and then use the Color selector button to change the color of the background.
