FlukeView 4.0 Combiscope Acquisition Software

The TS-400 Firmware Upgrade software is a user support interface for iTrack’s positioning and autonomous navigation systems. The software provides features for upgrading firmware, operating, configuring and monitoring the tracking system. Connectivity is provided through CAN (Intrepid ValueCAN, or Kvaser Diagnostic Cable), USB or Wireless Network (Wi-Fi).
